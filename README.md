# Projet-MI1-E-Semestre-2
# Creative-Yumland 🍽️

Projet Web (préING2 • 2025-2026) : création d’un site de restaurant / chaîne de restaurants avec une application **multi-utilisateurs** couvrant toute la chaîne de commande : **choix client → commande → paiement → préparation → livraison → notation**.

## Objectif
Développer une application web permettant :
- au **client** de consulter la carte et commander (sur place / à emporter / livraison),
- à l’**administrateur** de gérer/contrôler les profils et le site,
- au **restaurateur** de gérer la préparation et le statut des commandes,
- au **livreur** de consulter les infos de livraison et clôturer la livraison.

## Profils utilisateurs
- **Client** (site)
- **Administrateur** (PC, accès global)
- **Restaurateur** (tablette, commandes à préparer / en livraison)
- **Livreur** (smartphone, interface simple adaptée petit écran)

## Pages prévues (Phase 1)
- Accueil (nom + recherche + plats mis en avant)
- Produits / Carte (recherche + filtres/catégories)
- Inscription
- Connexion
- Profil (infos, anciennes commandes, fidélité, édition “crayon” simulée en phase 1)
- Admin (liste utilisateurs)
- Commandes (restaurateur)
- Livraison (livreur + lien vers app de navigation)
- Notation (après livraison) 

## Phases du projet
- **Phase 1** : HTML/CSS (UI statique + charte graphique + documents PDF)
- **Phase 2** : PHP / serveur (stockage, inscription/connexion, etc.)
- **Phase 3** : JavaScript + DOM + requêtes asynchrones

## Stack (selon phase)
- HTML / CSS
- PHP
- JavaScript (DOM, async)
- Base de données MariaDB
